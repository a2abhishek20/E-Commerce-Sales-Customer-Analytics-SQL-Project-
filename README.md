# E-Commerce-Sales-Customer-Analytics-SQL-Project-
Overview  This project analyzes an e-commerce transactional database using SQL to extract key business insights related to revenue, customer behavior, product performance, and returns.  The goal is to simulate real-world Data Analyst tasks such as KPI calculation, ranking, revenue contribution analysis, and customer lifetime value (CLV).

🗂 Database Structure

The project includes 6 relational tables:
Customers
Products
Orders
Order_Items
Payments
Returns

🛠 SQL Skills Demonstrated

Complex JOIN operations
GROUP BY & HAVING
Aggregate functions (SUM, COUNT, AVG)
Subqueries
Window functions (RANK, Running Total)
Revenue & KPI calculations

📈 Key Insights Generated

Total Revenue & Average Order Value
Top 3 Customers by Revenue
Revenue Contribution by Category (%)
Customer Lifetime Value (CLV)
Return Rate (%)
Most Used Payment Method
Monthly Revenue Trend

💼 Business Impact

This project demonstrates the ability to transform structured data into actionable business insights using SQL, replicating real-world Data Analyst responsibilities.
